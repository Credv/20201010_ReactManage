
# 20201010_ReactManage

#### 介绍
基于React16+Antd4.0开发的中后台管理系统


#### 安装教程

1.  Fork 本仓库
2.  Fork 服务端 https://github.com/Credv/20201010_ReactManage_Server.git
3.  下载mongodb

#### 使用说明
1.  安装依赖 npm i
2.  运行服务器 npm start
3.  运行项目 npm start
